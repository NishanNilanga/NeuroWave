# NeuroWave

AI-powered Music Visualizer with Hand Gesture Control

## Features
- Audio Reactive Visualization
- MediaPipe Hand Tracking
- Play / Pause with Gestures
- Zoom In / Zoom Out Controls
- Real-time Particle Effects
- React + Three.js

## Tech Stack
- React
- Vite
- Three.js
- React Three Fiber
- MediaPipe
- Web Audio API
